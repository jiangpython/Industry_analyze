🚀 金融分析系统

智能金融分析系统 - 专注于医药、新能源、半导体、芯片等行业的数据采集、处理和分析

📊 实时数据监控

获取股票实时价格、成交量、涨跌幅等实时市场数据，支持A股所有股票代码，智能缓存机制提升响应速度。

📈 历史数据分析

查看股票历史价格、K线数据、财务数据等历史信息，支持日线、周线、月线数据，提供历史趋势分析。

🏢 公司智能分析

公司基本信息管理、财务数据分析、AI智能分析报告、公司对比分析、风险评估等功能。

🏭 行业深度研究

行业数据、行业公司列表、行业趋势分析、行业对比研究、投资机会分析等功能。

⚙️ 任务智能管理

数据采集任务管理、分析任务调度、任务状态监控、任务结果查看、自动化任务配置。

🔌 多数据源支持

AKShare、Yahoo Finance等多数据源支持，数据源配置管理、数据质量监控、数据源切换功能。

🎯 快速开始

  • API文档: 访问 /docs 查看完整的API文档
  • API概览: 访问 /api/v1/ 查看所有可用端点
  • 系统状态: 访问 /health 检查系统健康状态
  • 实时数据: 使用 /api/v1/realtime/stock/{股票代码} 获取实时数据
  • 公司信息: 使用 /api/v1/companies/ 获取公司列表
  • 行业数据: 使用 /api/v1/industries/ 获取行业信息

📋 API端点概览

📊 实时数据
GET /api/v1/realtime/stock/{symbol} 获取个股实时数据
GET /api/v1/realtime/companies/{industry} 获取指定行业的公司列表（实时）
GET /api/v1/realtime/cache/info 获取缓存信息
DELETE /api/v1/realtime/cache 清除缓存
📈 历史数据
GET /api/v1/historical/stock/{symbol} 获取股票历史数据
GET /api/v1/historical/financial/{symbol} 获取财务历史数据
🏢 公司管理
GET /api/v1/companies/ 获取公司列表
GET /api/v1/companies/{code} 获取公司详细信息
POST /api/v1/companies/{code}/analyze AI智能分析公司
GET /api/v1/companies/{code}/financial-data 获取公司财务数据
🏭 行业管理
GET /api/v1/industries/ 获取行业列表
GET /api/v1/industries/{industry} 获取行业详细信息
GET /api/v1/industries/{industry}/companies 获取行业公司列表
⚙️ 任务管理
GET /api/v1/tasks/ 获取任务列表
POST /api/v1/tasks/ 创建新任务
GET /api/v1/tasks/{task_id} 获取任务详情
🔌 数据源
GET /api/v1/data-sources/ 获取数据源列表
GET /api/v1/data-sources/{source}/status 检查数据源状态
